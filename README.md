### Key Python Concepts Practised
+ **Counter**\
  -Counts occurrences, performs arithmetic on counts\
  -Applicable for: Word frequency, product popularity, inventory tracking
+ **deque**\
  -'Double-ended queue' used to quickly add/remove from both ends of a list using append(), appendleft(), pop(), and popleft()\
  -Applicable for: Assessing activity/log recency, task scheduling with priority, checking for palindromes
+ **namedtuple**\
  -Creates structured sequences with named fields\
  -Applicable for: Reading rows from CSV files/database records, storing item details
+ **defaultdict**\
  -Automatically creates a dictionary key if it doesn't exist yet, removing the need to check if it exists first before adding it\
  -Applicable for: Cleaner grouping of data, aggregation
+ **lambda**\
  -Creates an anonymous function (one without a name) on the fly\
  -Applicable for: Simplifying short-term tasks, reducing clutter when defining temporary/throwaway code
